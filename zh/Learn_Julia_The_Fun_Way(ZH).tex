\hypertarget{sec:setup_julia_environment}{%
\section{配置 Julia 开发环境}\label{sec:setup_julia_environment}}

本书所使用的 Julia 版本为 1.6.1
，建议读者阅读本书时使用该版本或者是更新的版本。

这一章主要介绍如何配置基本的 Julia 开发环境，以及一些常见的开发工具。

\hypertarget{ux672cux5730ux5b89ux88c5}{%
\subsection{本地安装}\label{ux672cux5730ux5b89ux88c5}}

官方推荐的安装方式是，前往\href{https://julialang.org/downloads/\#current_stable_release}{下载}页面，根据你当前的操作系统，下载对应的预编译好的文件，解压后将其中的
\passthrough{\lstinline!bin!}
子目录添加到\textbf{环境变量}中，然后重启命令行，执行
\passthrough{\lstinline!julia!} 即可进入
\href{@sec:REPL}{\passthrough{\lstinline!REPL!}} 。

不过国内其实已经有不少高校提供了镜像资源，在网络环境不太好的时候，大家可以直接从镜像站下载。以下是一些常见
Julia 二进制文件的镜像链接：

\begin{itemize}
\tightlist
\item
  \href{https://mirrors.bfsu.edu.cn/julia-releases/bin/}{bfsu}
\item
  \href{https://mirrors.tuna.tsinghua.edu.cn/julia-releases/bin/}{tuna}
\item
  \href{https://mirrors.sjtug.sjtu.edu.cn/julia-releases/bin/}{sjtug}
\item
  \href{https://mirrors.ustc.edu.cn/julia-releases/bin/}{ustc}
\item
  \href{https://opentuna.cn/julia-releases/bin/}{opentuna}
\end{itemize}

当然，有一些脚本可以帮我们地完成上面的步骤，这在某些自动化的流程中非常有用。

\begin{itemize}
\tightlist
\item
  \href{https://github.com/abelsiqueira/jill}{jill.sh}
\item
  \href{https://github.com/johnnychen94/jill.py}{jill.py}
\item
  \href{https://github.com/songxianxu/jvc}{jvc}
\end{itemize}
